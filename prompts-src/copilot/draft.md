# Stage 1: Draft Plan (Copilot)

Create an initial implementation plan from the preflight specification.

## Output Contract

Output exactly these sections in order:

## Goal and Success Criteria

Summarize the objective and completion conditions.

## Scope and Non-Goals

List what will be changed and what will not be changed.

## Technical Approach

Describe architecture decisions and implementation strategy.

## Step-by-Step Execution Plan

Numbered sequence of concrete implementation steps.

## Verification Plan

Commands and checks to validate success.

## Risks and Mitigations

List major risks and practical mitigations.

## Open Questions

List unresolved points. If none, write `None.`

## Rules

- Do not ask clarifying questions.
- Do not output code diffs.
- Keep the plan actionable for direct implementation.
